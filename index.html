In your own words, what is the main job of CSS? ANSWER: The main job of CSS is to control the visual presentation and layout of a webpage, such as its colors, fonts, and spacing. ====================================================================== 9. What does CSS stand for? ANSWER: Cascading Style Sheets. ====================================================================== 10. A developer can put CSS in a separate CSS file. Why might this be helpful when building a website? ANSWER: It keeps the code organized, allows you to change the look of an entire website by editing just one file, and helps pages load faster. ====================================================================== PART 3: JAVASCRIPT CONNECTION 11. In your own words, what can JavaScript do? ANSWER: JavaScript adds dynamic behavior and interactivity to a website, like pop-up alerts, drop-down menus, and updating content without reloading the page. ====================================================================== 12. Are JavaScript and Java the same language? ANSWER: No, they are completely different programming languages with different purposes, uses, and syntax. ======================================================================
